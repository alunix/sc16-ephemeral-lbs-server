Welcome to the help page!

Here you will get some help and hints for using the website!

Accessing messages

First you have to choose a zone that is shown within the map. After choosing one the messages are displayed in the message window on the right and ordered from newest to oldest.

Creating zones

For creating a zone you have to use the "..." button. After activating it you can create a region directly on the map and set its properties afterwards. It is also important to set a duration time for the zone.

Exploring statistics

The website offers statistics the show the activity in each zone. For exploring the statistics you have to choose a zone in the map and click the "..." button.
